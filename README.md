
# Java Mini Projects Repository

This repository contains multiple Java projects demonstrating **fundamental programming concepts**, **object-oriented programming**, **algorithms**, and **interactive console applications**. These projects are ideal for beginners and intermediate learners to understand **core Java**, problem-solving, and project structuring.

---

## **Projects Included**

### 1. Random Number Guessing Game
- **File:** `Random_number.java`
- **Description:**  
  An interactive number guessing game where the user has 10 attempts to guess a randomly generated number between 1 and 100. The game provides hints if the guess is too high or too low and calculates points based on attempts.
- **Key Concepts:** Loops, Conditional Statements, Random Numbers, User Input.
- **How to Run:**
```bash
javac Random_number.java
java Random_number
````

---

### 2. Sudoku Solver

* **File:** `Sudoku.java`
* **Description:**
  A console-based Sudoku solver using **backtracking algorithm**. It can solve any standard 9x9 Sudoku puzzle by filling in empty cells according to Sudoku rules.
* **Key Concepts:** Recursion, Backtracking, 2D Arrays, Logic Implementation.
* **How to Run:**

```bash
javac Sudoku.java
java Sudoku
```

---

### 3. Exam Portal (Login & Signup)

* **File:** `Test.java`
* **Description:**
  A simple exam portal simulation where users can **sign up**, **login**, and attempt a 10-question English vocabulary exam. Scores and performance feedback are provided at the end.
* **Key Concepts:** Classes & Objects, Methods, User Input, Conditional Logic.
* **How to Run:**

```bash
javac Test.java
java Test
```

---

### 4. Bank Management System

* **File:** `bank.java`
* **Description:**
  A console-based banking system that allows multiple users to **deposit**, **withdraw**, **transfer funds**, and view **transaction history**. Demonstrates basic account management logic and OOP in Java.
* **Key Concepts:** Object-Oriented Programming, Arrays, Lists, Date & Time, User Interaction.
* **How to Run:**

```bash
javac bank.java
java bank
```

## **Setup Instructions**

1. Clone the repository:

```bash
git clone https://github.com/SreyaThota/Java-Mini-Projects.git
```

2. Navigate to the project folder.
3. Compile the `.java` file for the project you want to run:

```bash
javac <filename>.java
```

4. Execute the program:

```bash
java <classname>
```
